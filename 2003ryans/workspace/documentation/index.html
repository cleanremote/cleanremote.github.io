Documentation

Windows Shortcuts


Alt + F4 - send Alt + F4 (closes windows)
Alt + Tab - send Alt + Tab (switches windows)
Delete - send Delete
Enter - send Enter
Ctrl Alt Delete - send Ctrl Alt Delete
Start Menu - open Start Menu

Annoying


System Shutdown - Turns off computer
System Reset - Immediately resets computer
NMI (Force BSOD) - Forces a blue screen of death (this does not go away)
Enable Single-Step Mode - Single Step Mode enables QEMU in single step mode. It will crash the computer and make the computer extremely slow on the next boot.
Stop Computer - This will completely stop the computer.
Eject Windows XP CD - If the computer is not restoring snapshots, this will prevent people from reinstalling Windows if the hard disk breaks.
Rick Roll - This will spam the first two lines of the song "Never Gonna Give You Up".

Very Annoying


Turn Bomb - This will flood the server with false turn requests several times and will rig the turns up to ~800 seconds.
Super Turn Bomb - This will flood the server with several thousand false turn requests several times and will rig the turns up to ~8849 seconds.

Other


Continue Computer - If you clicked on "Stop Computer", this will make the computer's state resume.
Insert Windows XP CD - You can reinsert the Windows XP CD if needed.
Fix Turn Bomb - Sends a command to quit the emulator which will erase all coexisting turns which clears a turn bomb. It may also be useful for annoying people.
System Wakeup - Used if somebody enters Stand By mode. (If it doesn't work, use Fix Turn Bomb)

Hard Disk Functions


Set Boot to Hard Drive - Sets booting to hard disk.
Set Boot to Floppy - Sets booting to floppy disk.
Set Boot to CD - Sets booting to CD. (Used for reinstalling Windows if snapshots are broken.)
Set Boot to Network - Sets booting to Networking, or iPXE terminal.
